# JTrash

**JTrash** is a simple card game developed for PC in Java, using JavaFX for graphics. It was created as an exam project, focusing on game mechanics, user interface, and animations.

### Project Overview

JTrash is a digital adaptation of the classic card game *Trash*, designed to provide an engaging and user-friendly experience with smooth animations and AI opponents.

---

### Responsibilities

As the sole developer of this project, I was responsible for:

- **Game Logic**: Implementing the rules and mechanics of the *Trash* card game.
- **User Interface**: Designing and developing an intuitive UI with JavaFX.
- **Animations**: Creating smooth visual transitions for gameplay.
- **AI Implementation**: Developing an efficient AI to play against human users.
- **User Input Handling**: Ensuring smooth interactions between the player and the game.

---

### Technologies Used

- **Programming Language**: Java
- **Graphics Library**: JavaFX

---

### Challenges Faced

- **Low-Level Rendering**: Optimizing animations and visual effects with JavaFX.
- **Game Rule Implementation**: Ensuring all possible gameplay scenarios were correctly handled.
- **UI/UX Design**: Making the interface user-friendly and visually appealing.
- **Efficient AI Development**: Creating a competitive AI opponent that provides a challenge without feeling unfair.

---

### Outcome

The project successfully resulted in a fully functional digital version of *Trash*, featuring:

- **Smooth and Interactive Gameplay**: Players can easily pick up and play the game.
- **Well-Designed UI**: A clean and user-friendly interface.
- **AI Opponent**: The game includes a computer-controlled opponent for solo play.
